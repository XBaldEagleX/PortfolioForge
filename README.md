# PortfolioForge 🚀

Automatically builds a stunning personal portfolio website + PDF résumé from your GitHub and LinkedIn data.

**One command → live portfolio at `yourusername.github.io`**

## Features
- Pulls all your GitHub repos, stars, languages
- Parses LinkedIn PDF export
- AI-powered project summaries & About section
- Beautiful responsive HTML site (dark/light mode)
- Professional PDF résumé
- One-click deploy to GitHub Pages

## Quick Start
```bash
pip install -r requirements.txt
cp .env.example .env     # then edit .env
python main.py
